Install
*******

Using tourbillon
================


To install tourbillon-nginx at a command prompt type: ::

	$ sudo -i tourbillon install tourbillon-nginx



Using pip
=========


To install tourbillon-nginx using pip enter: ::

	$ pip install tourbillon-nginx

.. note::
	
	If you have installed tourbillon in a virtualenv you must install tourbillon-nginx in the same virtualenv.
